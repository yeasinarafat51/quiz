Step 1: Basic HTML Questions 1. What is HTML? Explain its purpose & Basic Syntax Of HTML ? answer: The primary purpose of HTML is: Structure: It organizes web content into headings, paragraphs, lists, tables, and more. Content Semantics: Tags provide meaning to content, helping search engines and browsers understand what each section is about. Linking: HTML enables links (hyperlinks) to other pages, creating a navigable web structure. Embedding Media: It supports adding images, videos, and other multimedia.
